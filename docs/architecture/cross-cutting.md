| Module | Purpose | 접근 가능 레이어 | 디렉토리 |

## 사용 규칙

1. 횡단 모듈은 **다른 횡단 모듈을 참조할 수 있음** (단, 순환 금지)
2. 횡단 모듈은 **비즈니스 레이어를 참조할 수 없음** (Service, Repository 등)
3. 새 횡단 모듈 추가 시 이 문서에 등록 필수

## 새 횡단 모듈 추가 기준

다음 조건을 모두 만족해야 횡단 모듈로 분류합니다:

- [ ] 3개 이상의 레이어에서 사용됨
- [ ] 특정 비즈니스 도메인에 종속되지 않음
- [ ] 독립적으로 테스트 가능
