## 문서 작성 규칙

1. **날짜 기입** — 문서 상단에 작성일/수정일 기재
2. **교차 링크** — 관련 문서는 상대 경로로 링크
3. **검증 상태** — 결정 사항에는 검증 상태 (✅ 검증됨 / ⚠️ 실험 중 / ❌ 폐기) 표기
4. **에이전트 가독성** — 에이전트가 파싱할 수 있도록 테이블, 목록 등 구조화된 형식 사용
